# Product

A Discord self-bot that automatically joins a specific voice channel, stays muted and deafened, and reconnects automatically if moved or disconnected.

## Key Behaviors
- Joins a configured voice channel on startup
- Maintains self-mute and self-deaf state
- Detects channel moves and preserves the connection
- Auto-reconnects on disconnection
- Periodic keep-alive loop to monitor connection health

## Important Warning
This project uses a Discord user token (self-bot), which violates Discord's Terms of Service. It is intended for educational purposes only.
